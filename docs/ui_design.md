# 整体风格

- 遵循 macOS 原生设计风格
- 使用简洁、轻量的视觉元素
- 采用适当的留白和间距
- 图标使用 awesome 图标库

# 窗口尺寸
- 建议采用紧凑型设计
- 支持响应式布局
