百词斩助手 Web

功能使用说明文档

核心功能

功能一

选词翻译

在浏览网页时，只需选中任意单词，即可立即弹出该单词的详细释义窗口。无需跳转页面，一键查词，让英语阅读更加顺畅。

功能二

单词搜索

内置强大的搜词功能，输入单词即可快速查询单词的完整释义、音标、例句等信息。支持模糊搜索，帮助您快速找到想了解的单词。

功能三

收藏同步

遇到生词一键收藏，数据自动同步到百词斩APP，实现Web端与移动端的无缝衔接，随时随地复习你收藏的单词。

功能四

Anki 导出

支持将收藏的单词批量导出为Anki卡片格式，深度整合Anki复习系统，使用您熟悉的工具进行个性化学习。

功能五

在线学习

直接在浏览器中进行单词学习和复习，采用卡片式学习模式，支持键盘快捷键操作，搭配自动音频播放，打造沉浸式学习体验。

技术栈

React TypeScript Tailwind CSS Zustand React Router Vite Axios
